    Portfolio Tracker

This project is somewhat influenced by my mother as she has taken a recent liking to watching economics and finance videos, and has begun investing in mutual funds herself through CIBC. This project uses the standard web-scraping libraries (BS4, requests), standard data visualization libraries (Pandas, Numpy, Matplotlib), and miscellaneous other libraries (datetime, timezone). The main library is YFinance, and it is based upon the aforementioned libraries, as it draws upon Yahoo Finance's data.

This will be my first solo project of this scale, it will be a continuous project until I am satisfied with its completed state. Final product will be solely for personal use.
