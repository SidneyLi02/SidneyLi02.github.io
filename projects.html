HACKMIT2020: SafeT
HTML, CSS, JS, Google Maps API

Created and pitched prototype webapp at HackMIT with freshmen Shang and Lin. SafeT uses COVID-19 data published by the City of Toronto and an interactive Google Maps map to safely plan the user's walking route from a chosen Point A to Point B within Toronto. interactive map, heatmap, and routing were made possible through Google Maps API or related Google APIs.

Financial Tracker
Python, Pandas, BeautifulSoup4, Requests, Matplotlib, Yfinance

Python webscraper which utilizes Requests and BeautifulSoup4 to scrape the daily top gainers and losers from their respective Yahoo Finance webpage, and downloads into a Pandas dataframe, wherein it allows the user to plot the stock's price over a period of time through Matplotlib.

Coca-Cola Piano
Java, Arduino Uno R3, CapacitativeSense

Created an 8-key Coca-Cola piano with Wen and Lee which utilizes Arduino' CapacitativeSense library to determine a user's touch on a Coca-Cola can and plays the respective frequency through a piezo buzzer. The project required usage of jumper wires, a breadboard, a set of batteries, a piezo buzzer, along with various other circuitries, and knowledge of electrical circuits and voltage.
